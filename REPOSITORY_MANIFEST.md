## Intentionally Not Included

These are generated locally and should not be committed to GitHub:

- `node_modules/`: reinstall with `npm install`
- `.next/`: generated by `npm run dev` or `npm run build`
- `out/`: static export generated by `npm run build`
- `.firebase/`: Firebase CLI cache generated during deploy
- `next-env.d.ts`: generated by Next.js
- `tsconfig.tsbuildinfo`: generated by TypeScript

## Recreate Generated Files On Another Machine

```bash
git clone https://github.com/netihatthaya11-bit/Neti-Hatthaya.git
cd Neti-Hatthaya
npm install
npm run build
```

After `npm run build`, the `out/` folder will be recreated for Firebase Hosting.
